Use this Calculator to figure the total costs incurred during an individual patient’s ICU stay. Each session will include data for that patient’s ICU stay; new sessions will be added to the list of sessions below. Clicking the Create New Session button starts a new session. A previous session can be restarted by selecting its name from the list below and clicking on the action that you wish to perform (edit, copy, rename, delete). After saving a session, you can return to it later.

For each agent in a patient’s sedation regimen, enter the agent’s concentration (mcg/mL or mg/mL), its vial size (in mL), and the cost per vial; then click Next.
